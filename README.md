# Modelling and Fabrication Techniques - Master's Class Website

A modern, responsive static website for displaying information about the Master's level course on Modelling and Fabrication Techniques, offered by **FabLab Rwanda** to **UR-ACEIoT** (University of Rwanda - Advanced Centre for Internet of Things).

## Features

- **Modern Design**: Clean, professional interface with smooth animations
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Interactive Elements**: Hover effects and fade-in animations
- **Well-Organized Sections**:
  - Course Overview
  - Course Topics (6 main topics)
  - Learning Outcomes
  - Course Details (Duration, Credits, Format, Assessment)
  - Prerequisites
  - Contact Information

## Files Structure

```
├── index.html      # Main HTML structure
├── styles.css      # All styling and responsive design
├── script.js       # Interactive features and animations
└── README.md       # This file
```

## Usage

Simply open `index.html` in any modern web browser. No build process or server required - it's a fully static website.

## Customization

You can easily customize:

- **Colors**: Modify CSS variables in `styles.css` (root section)
- **Content**: Edit the HTML content in `index.html`
- **Animations**: Adjust animation timings in `script.js` and `styles.css`
- **Fonts**: Change the Google Fonts import in `index.html`

## Browser Support

Works on all modern browsers:

- Firefox (latest)
- Safari (latest)
- Mobile browsers
